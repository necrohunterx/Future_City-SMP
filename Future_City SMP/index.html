HOW TO PLAY?

Play the SMP the same way you play Minecraft normally. Here you will get a diamond enchanted armor[Protection 2, Thorns 1, mending, Unbreaking 2], diamond enchanted weapons[Power 3, Punch 1, Mending, Unbreaking 2] and some diamond enchanted[Mending, Unbreaking 2] tools. You will have jobs and salary according to the jobs. For the jobs you can contact the operator [Sri Charan]. Here you can chat and play Minecraft together with players. The server can be joined by Java/Bedrock, and version is 1.20.4 only. For joining the server you need to be whitelisted. So for joining the server and getting the access in aternos to start the server you can contact the operator [Sri Charan].
